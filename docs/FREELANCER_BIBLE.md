## The Job Workflow (Trades Worker)

```
1. SITE VISIT       → Customer calls, they visit the site
2. ESTIMATE         → Labour + materials breakdown
3. APPROVAL         → Customer confirms
4. WORK ORDER       → Job begins, track progress
5. MATERIAL PURCHASE→ Buy materials, record expense
6. HELPERS / TEAM   → Daily wage workers + partner splits
7. COMPLETION       → Work done, final invoice raised
8. PAYMENT          → Advance + final collection
```

### Profit Per Job
```
Job Value:        ₹50,000
- Materials:      ₹18,000
- Helper wages:   ₹4,000
- Partner share:  ₹15,000
─────────────────────────
My Profit:        ₹13,000
```

## The Growth Path

```
Stage 1 — Solo
Works alone, small jobs
Needs: Quotes + Invoices + Get paid

Stage 2 — With Helpers
Takes bigger jobs, brings daily workers
Needs: Jobs module + Helper wages + Profit per job

Stage 3 — Small Team
3-4 permanent staff, regular clients
Needs: Staff attendance + Salaries + Multiple jobs

Stage 4 — Proper Business
Multiple teams, big contracts, GST registered
Needs: Full ERP → upgrade business type
```

Syllabrix grows with them. They never need to leave.

## Design Philosophy

### 1. Zero Confusion
- Never show two buttons when one is enough
- Never use accounting language
- Speak their language always

### 2. Zero Dead Ends
- Every screen tells them what to do next
- Never show an error without a solution

### 3. Zero Setup
- Open app → start using immediately
- No configuration, no tutorials

### 4. Feels Like WhatsApp
- Familiar patterns
- Simple lists not complex tables
- Completion indicators

### 5. Automated By Default
- Payment reminders send themselves
- Monthly summary generates itself
- Tax estimate calculates itself

## The 3-Tap Rule

Any action must be completable in 3 taps or less.
```
Send a bill        → 3 taps
Record an expense  → 2 taps
Check who owes me  → 1 tap
Send a reminder    → 1 tap
```

## Automation Layer

### Automatic (Zero effort)
- Payment reminder on WhatsApp after 7 days
- Monthly earnings summary on 1st of month
- "You have 3 pending payments" alert
- Tax estimate updated every month

### Semi-Automatic (One tap)
- "Want to send a reminder to Sharma Ji?"
- "Job complete? Want to raise the final bill now?"
- "This client pays late. Ask for advance next time?"

### Smart Suggestions (App learns)
- "You usually charge ₹500/hr. Use same rate?"
- "Last time Mehta Ji took 45 days to pay."
- "You bought wire from this shop 3 times. Save as supplier?"
