# Syllabrix Marketing Site — UI Kit

Public marketing site recreation. Single page (`index.html`) demonstrating the brand's hero, modules grid, AI video promo callout, pricing, and footer.

## Sections

| Component | What it is |
|---|---|
| `Header.jsx` | Logo, nav, sign-in, Get started CTA. |
| `Hero.jsx` | "Run any business. Beautifully." Headline, sub, business-type chips, screenshot frame. |
| `Modules.jsx` | Grid of business types with adapted copy. |
| `AiPromo.jsx` | Inverse callout for the AI video feature. |
| `Pricing.jsx` | Three plans — Starter / Growth / Scale. |
| `Footer.jsx` | Links, language, copyright. |
