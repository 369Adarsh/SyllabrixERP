# Syllabrix Mobile — UI Kit

Owner-facing mobile app. Three core screens in `index.html`, switchable via the bottom tabs:

1. **Home** — daily summary, KPI cards, quick actions, today's appointments.
2. **POS / Quick bill** — barcode scan, cart, total, save & share.
3. **AI Studio** — generate a promotional reel from live business data.

## Components
| File | What it is |
|---|---|
| `ios-frame.jsx` | iPhone bezel + status bar (starter component). |
| `MobileShell.jsx` | App shell — top bar + bottom tab bar, tab switcher. |
| `HomeScreen.jsx` | KPI cards, quick actions, today list. |
| `PosScreen.jsx` | Scan bar, cart, totals, primary action. |
| `AiStudioScreen.jsx` | Goal picker, template grid, render button. |
